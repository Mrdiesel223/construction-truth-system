## ✅ Completed: Backend API (`apps/backend`)
A robust REST API with the following features:
- **Authentication:** JWT-based Login & Registration with password hashing (Bcrypt).
- **Database Schema:** 
    - `User`: Roles (Admin/Staff), Credentials.
    - `Attendance`: GPS-tagged Start/End day logs.
    - `Visit`: Customer details, GPS coordinates, notes, and image references.
- **Middleware:** Secured routes using an `auth` middleware.

---

## ✅ Completed: Mobile App (`apps/mobile`)
A Flutter application with a focus on field usability:
- **State Management:** Provider pattern for Auth and Data.
- **Security:** `flutter_secure_storage` for keeping tokens safe.
- **Features:**
    - **Login Screen:** Secure entry point.
    - **Dashboard:** Quick actions for Start/End day with automatic GPS capture.
    - **Visit Entry Screen:** Full form for recording customer visits, including camera integration for "Proof of Visit."
- **Permissions:** Configured for high-accuracy location and camera access on both Android & iOS.
