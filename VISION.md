# 🏗️ Field Staff & Construction Intelligence System - VISION

## 1. Purpose
This system is designed for **construction contractors (bridges, buildings, site work)** to:
- Track worker attendance
- Assign and monitor tasks (curing, shuttering, etc.)
- Capture real-time work proof (photos, time, location)
- Reduce manual supervision
- Prepare data for future AI-based monitoring (camera + automation)

---

## 2. System Philosophy
This is NOT just a tracking app. It is a **truth system**:
- What was planned?
- What actually happened?
- What is verified?

---

## 3. Tech Stack
- **Mobile:** Flutter (Provider/Riverpod)
- **Backend:** Node.js + Express + Prisma (PostgreSQL)
- **Admin:** Next.js + Tailwind + ShadCN
- **Future:** Python + OpenCV + YOLOv8

---

## 4. Core Modules
- **Authentication:** Role-based (Admin/Staff/Manager).
- **Attendance:** GPS-tagged Start/End work.
- **Task Management (CRITICAL):** Admin assigns, Worker executes with proof.
- **Proof Capture:** Mandatory images/video at task completion.
- **Truth Logic:** No task ends without proof + location.

---

## 5. Deployment & Integrity
- **Final Principle:** Build clarity, accountability, and truth.
